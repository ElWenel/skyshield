# SkyShield Roofing - Project Images

This folder contains all image assets for the website.

## Required Images

### Hero Images

- `hero.webp` - Main hero section background (1200x800px)

### Portfolio Images

- `portfolio-1.webp` - Portfolio image 1 (800x600px)
- `portfolio-2.webp` - Portfolio image 2 (800x600px)

### Branding

- `logo.svg` - Company logo (SVG format recommended)
- `favicon.ico` - Browser tab icon (32x32px)

## Image Guidelines

**Format**: WebP preferred for best compression, PNG/JPG as fallback
**Optimization**: Use TinyPNG.com or ImageOptim for compression
**DPI**: 72 DPI for web (not print)
**Color Space**: RGB

## Replacing Images

1. Delete existing image file
2. Add new image with same filename
3. Ensure dimensions match requirements
4. Test on mobile and desktop

## Resources

- **Royalty-free images**: Unsplash.com, Pexels.com, Pixabay.com
- **Image conversion**: CloudConvert.com
- **Image compression**: TinyPNG.com
- **Design tool**: Canva.com
